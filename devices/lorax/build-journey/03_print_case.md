# Step 3: Print Case

## Files Required

- `enclosures/lorax_case.stl` (or `.step` for CAD)
- `enclosures/lorax_case_bottom.stl`

## 3D Printer Settings

### FDM (PLA/PETG)

- **Layer height**: 0.2mm
- **Infill**: 20-30%
- **Supports**: None (if printed upright)
- **Walls**: 3-4 perimeters
- **Temperature**: 220°C (PLA) or 240°C (PETG)

### Resin (SLA)

- **Layer height**: 0.05mm
- **Exposure**: Per resin spec (typically 2-3s)
- **Supports**: Light supports on overhangs
- **Orientation**: Tilt 45° to reduce layer lines

## Post-Processing

1. **Remove supports** carefully
2. **Sand** edges (320 → 600 grit)
3. **Test fit**: PCB should slide in smoothly
4. **Drill** antenna hole if needed (2.5mm)

## Assembly

1. **Install PCB** into case
2. **Secure** with M2 screws through standoffs
3. **Attach antenna** through case hole
4. **Snap** top and bottom halves together

## Optional: Waterproofing

- Apply silicone sealant to seams
- Use O-rings for antenna connector
- Test in shallow water before deployment
